## 程序结构、编译 & 执行
```python
# 导入必要的模块
import math
# 定义函数
def square_root(num):
    """计算一个数字的平方根"""
    return math.sqrt(num)
# 主程序入口
if __name__ == '__main__':
    # 获取用户输入
    num = float(input("请输入一个数字："))  
    # 调用函数进行计算
    result = square_root(num)    
    # 在控制台输出结果
    print(f"{num}的平方根是：{result}")
```
